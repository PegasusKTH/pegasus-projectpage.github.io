Our Project

Our project is a graph based approach towards building a collection of all courses in KTH and then cataloging them based on program and course specific prerequisites. It helps you navigate through the many courses at KTH so that you can easily understand which courses you need to take and plan out your studies.

Definition of done

When is a story done?

It should be tested with the pre-conditions and fulfill the post-conditions. There should be no unaccepted bugs (There are acceptable ones :) Check with team/product owner). All tasks regarding the story should be integrated.

When is a task done?

Tasks should be tested by at least one other group than the group working on the task. A task should be able to be integrated with other tasks in the same story. There should be no unaccepted task-specific bugs.
